# FitTrack Pro 2.0 — Инструкция по запуску

## Требования
- [Node.js](https://nodejs.org/) версии 18 или выше

## Быстрый старт

### 1. Установка зависимостей
```bash
cd fittrack
npm install
```

### 2. Запуск сервера
```bash
npm start
```

Сервер запустится на `http://localhost:3000`

Откройте браузер и перейдите по адресу **http://localhost:3000**

## Как работает система аккаунтов

- **Регистрация** — придумайте логин (мин. 3 символа) и пароль (мин. 4 символа)
- **Вход** — после входа все данные загружаются с сервера
- **Синхронизация** — данные автоматически сохраняются через 1.2 сек после изменений
- **Кнопка 🔄** — принудительная синхронизация
- **Кнопка 🚪** — выйти из аккаунта (данные остаются на сервере)
- **Между устройствами** — войдите с тем же логином/паролем на любом устройстве

## Переменные окружения (опционально)

| Переменная   | По умолчанию                          | Описание                  |
|-------------|----------------------------------------|---------------------------|
| `PORT`      | `3000`                                 | Порт сервера              |
| `JWT_SECRET`| `fittrack-secret-change-in-production` | Секрет для JWT-токенов    |

**Для продакшена обязательно задайте `JWT_SECRET`:**
```bash
JWT_SECRET=мой-секретный-ключ-123 npm start
```

## Доступ с других устройств в локальной сети

1. Узнайте IP вашего компьютера:
   - Windows: `ipconfig` → IPv4-адрес
   - Mac/Linux: `ifconfig` или `ip a`
2. Запустите сервер: `npm start`
3. На другом устройстве откройте: `http://ВАШ_IP:3000`
